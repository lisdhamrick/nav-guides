Standalone, no-logins viewer

Step Guide

Build step-by-step visual guides and share them with public links or embeds. No accounts required for viewers.

Create a guide View a guide

How it works

  1. Build your guide in the editor.
  2. Export JSON (self-contained with images).
  3. Publish JSON on GitHub Pages.
  4. Share or embed the viewer URL.

Embed friendly

The viewer works in an iframe. It can load a guide from a JSON file in your GitHub Pages repo.

viewer.html?src=guides/sample.json

PDF export

The viewer supports print styles. Use the "Print / Save PDF" button to export.

Optional media

Add video links or audio voiceover per step.
